# Diabetes Prediction using Machine Learning

This project predicts whether a patient has diabetes based on medical features.

## Dataset
Healthcare Diabetes Dataset from Kaggle.
https://www.kaggle.com/datasets/nanditapore/healthcare-diabetes

## Models Tested
- Logistic Regression
- Random Forest
- XGBoost
- Gradient Boosting
- KNN
- SVM

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

## Results
Random Forest and XGBoost achieved the best performance with an F1 score of ~0.99.
